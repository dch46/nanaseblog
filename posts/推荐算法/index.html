什么是推荐算法，意义，使用场景

  • 推荐算法是计算机专业中的一种算法，通过一些数学算法，推测出用户可能喜欢的东西，目前应用推荐算法比较好的地方主要是网络，其中淘宝做的比较好。
  • 帮助用户快速找到想要的商品，提高用户对网站的忠诚度；
  • 淘宝，抖音，B站等任何须有个性化定制的网站

启发式算法如何计算相似度

欧氏距离计算（计算两个点之间的直线距离） $$ |x| = \sqrt{ (x_1^2 + x_2^2 + … + x_n^2)} $$

import math
# 计算两点之间的距离
def eucliDist(A,B):
    return math.sqrt(sum([(a - b)**2 for (a,b) in zip(A,B)]))
X = [1,2,3,4]
Y = [0,1,2,3]
print(eucliDist(X,Y))
